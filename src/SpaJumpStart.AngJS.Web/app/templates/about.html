About

This is an example of a SPA app with Modals using Angular JS Framework.

The following JavaScript Frameworks were used:

  • Typescript

  • JQuery 2.1.4

  • Bootstrap 3.3.6

  • Angular JS - 1.4.7

  • Angular-Animate JS - 1.4.8

  • Angular-Route - 1.4.7

  • Angular-UI-Bootstrap - ~0.12.0

  • NodeJS - 4.2.3
